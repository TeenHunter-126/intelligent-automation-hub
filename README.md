# Muhammad Zia Ul Hassan — Portfolio

Personal portfolio website for **Muhammad Zia Ul Hassan**, Mechatronics & Control Engineer specializing in Robotics, Industrial Automation, Machine Learning, and Control Systems.

## Live Site

[mziahassan.com](https://mziahassan.com)

## Tech Stack

- **Framework:** [TanStack Start](https://tanstack.com/start) (React + SSR)
- **Routing:** [TanStack Router](https://tanstack.com/router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com) (New York style)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Build Tool:** [Vite](https://vitejs.dev)
- **Deployment:** [Cloudflare Workers](https://workers.cloudflare.com)
- **Language:** TypeScript

## Features

- Fully responsive, mobile-first design
- Dark space-themed aesthetic with glassmorphism UI
- Animated hero section with typewriter effect
- Skills, Projects, Experience, and Education sections
- Volunteer & community involvement section
- Contact section with direct links
- CV download
- SEO optimized with Open Graph, Twitter Card, and JSON-LD structured data

## Project Structure

```
src/
  assets/          # Static assets (portrait image)
  components/
    portfolio/     # Page sections (Hero, About, Skills, Projects, etc.)
    ui/            # shadcn/ui component library
  hooks/           # React hooks
  lib/             # Utility functions
  routes/          # TanStack Router file-based routes
  styles.css       # Global styles and design tokens
  router.tsx       # Router configuration
public/
  zia-cv.pdf       # Downloadable CV
  favicon.svg      # Site favicon
  site.webmanifest # PWA manifest
```

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
